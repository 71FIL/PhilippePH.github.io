Web Security Recommendations

Make your updates: This is one of the best security practices there are. Browser updates takes only one or two seconds, and could be fixing important vulnerabilities.

Password management:

  1. Use a password manager
  2. Use password best practices (strong passwords, no repetition, etc.)
  3. Do not save your passwords (or credit card information) on your web browser. Use your password manager instead.

Two-factor authentication (2FA):

  1. Always enable 2FA, when possible.
  2. For regular users, I'd recommend using authenticator apps. Avoid SMS options. For more advanced users, have a look at YubiKeys.

Browser the web using HTTPS: HTTPS encrypts requests and responses made between your device and the webistes you visit. Make sure your browser is set up to enable HTTPS-Only (most browsers have this functionality). Alternatively, you can use extensions such as HTTPS Everywhere.

These recommendations are not exhaustive, please treat them as suggestions.
