My browser privacy setup recommendation

Not sure why you should care yet? Check this page out: The Relevance of Web Privacy for Everyday Users

  • Use a more privacy-focused browser. I currently use Firefox.
  • Install the Ghostery web extension. It will help block a lot of trackers, and reduce the number of annoying cookie banner.
  • Install the containers extension, and set it up according to your internet usage. This isolates cookies, scripts, etc. per container, meaning you "containarize" your browsing ** ADD HERE ***. This is also great for security reasons, as malicious scripts are limited to the container they are contained in.
  • Disable sponsored shortcuts and sponsored stories (I disable them altogether). I also disabled Snippets. (here)
  • Change the default search engine to DuckDuckGo (here)
  • Use "custom" tracking protection. Tick every box, and select "all cross-site cookies" and "in all windows". (here) Personally, I have found that very few websites actually break. In those rare cases, I simple click the Firefox lock icon in the search bar, disable enhanced protection for that one website, reload and then the issue is fixed.
  • I do NOT delete cookies and site data when Firefox is closed, because I have found this to be too much of a hurdle (it logs you out of every website, everytime Firefox closes). I do however enable the functionality "Clear history when Firefox closes" as this does not impact my browsing experience much.

These recommendations are not exhaustive, treat them as suggestions. Many thanks to Sun Knudsen. His great Youtube channel piqued my interest concerning web privacy and security, and a lot of my set up is based on his.

Do you absolutely want to keep using Google Chrome?

If you absolutely want to keep using Google Chrome (which I don't recommend), then here's how you can make some improvements to your setup.

  • Consider using a different web browser
  • Install the Ghostery web extension. It will help block a lot of trackers, and reduce the number of annoying cookie banner.
  • Disable everything in the ad menu
  • Remove "Make searches and browsing better"
  • Refuse when websites ask to access your location, unless you judge it to be absolutely necessary
  • Use the DuckDuckGo (or other privacy-focused search engine). For day-to-day browsing, DuckDuckGo should be satisfactory. When doing advanced research, it might be useful to use Google as its search results are still better.

These recommendations are not exhaustive, and I'd argue that you cannot really be private when using Google Chrome
